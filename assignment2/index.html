Assignment 2: Leaflet Playground

    Leave trace

1. View a marker's coordinates (excl. circle) by hovering over the marker (coordinates are added as the marker's title).

2. Generate random markers. The button "Generate random marker" creates a new marker at random coordinates within the bounds of Colorado, removes any (randomly generated) marker that might already be on the map and adds the newly generated marker.

3. When the checkbox "Leave trace" is checked, then a circle marker will be added to the map in the place of the last random-marker, before each addition of a new random-marker. Its popup will also be opened.

4. Press any key while on the map to report the number of traced (circle) markers:   To be printed here  

5. Make a marker the favorite. You can make a marker the favorite (that is, change its icon) by clicking on it, which calls the function "-=-=-". This removes any previous favorite and makes the clicked marker the favorite. Also, you can pan to the last favorite by double-clicking on the map. [Double-click again to pan to the current favorite.]

6. Reset to original view and erase all markers by clicking on the button right below the map.
