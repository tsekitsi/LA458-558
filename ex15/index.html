Exercise 15: Loading GeoJSON

Part 1. (i) (Turf.js) along: can be useful for an application where you want to have checkpoints along a line every few units of distance. (ii) (Turf.js) lineToPolygon: useful in case you need to work with properties of polygons (eg area) but you have a collection of lines instead, for the area of interest. (iii) (Turf.js) squareGrid: can help with analysis of different types of data, eg raster data. (iv) (Turf.js) convertArea: useful when you need to convert units for input to a different part in your code. (v) Proj4js: useful when you need to convert coordinate systems to match with the rest of your data.
